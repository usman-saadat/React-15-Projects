🚀 From Calendar App to Personal Blog – Heavy React Apps & Websites

This repository is a comprehensive showcase of 15 hands-on React projects that I built to master front-end development. Starting with a Calendar App and ending with a Personal Blog and Chatbot, each project progressively introduced new concepts, hooks, patterns, and integrations.

What I learned & practiced across these projects:

Core React concepts: components, props, state, and JSX
Hooks: useState, useEffect, useContext, useReducer, useRef, useCallback, useMemo
Routing with React Router DOM
State management (Context API + useReducer)
Form handling & validation
API integration (fetch, axios, error handling, loading states)
Authentication & protected routes
Local storage & data persistence
Custom hooks (reusable logic)
Component composition & conditional rendering
Styling: CSS modules, Tailwind CSS, Styled Components
Dark mode theming
Search, filter, sort, pagination
CRUD operations (create, read, update, delete)
Real-time features (Chatbot with basic AI logic)
Markdown rendering & rich text editing (Personal Blog)
Deployment (Vercel / Netlify)
Git workflow & project structuring
